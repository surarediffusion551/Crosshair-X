## Troubleshooting
| Problem | Solution |
|--------------------------------|-------------------------------------------------------|
| Overlay doesn't appear | Run as Administrator and make sure game is in borderless windowed mode |
| Hotkeys not responding | Enable administrative keyboard permissions in app settings |
| Crosshair misaligned / off-center | Adjust crosshair alignment sliders or recalibrate resolution |
| Exclusive presets missing | Update to latest version 2026 and re-sync community share |
| Windows 11 compatibility error | Update Windows and run the setup guide again |
| Game crashes when opening menu | Verify game overlay settings and clear Crosshair X cache |
| Download / update failed | Use the latest release and check your internet connection |
